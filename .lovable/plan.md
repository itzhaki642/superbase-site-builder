# נאור אדיר בע״מ - אתר קטלוג דלתות קירור

אתר תדמית וקטלוג מודרני-תעשייתי בעברית (RTL) עבור נאור אדיר בע״מ - מכירת דלתות קירור ומתן שירות לחדרי קירור ללקוחות תעשייתיים, עם פאנל ניהול מאובטח לעדכון תוכן.

---

## 🎨 שפה ויזואלית

- **פלטת צבעים**: אפור בטון כהה (#1f1f1f, #3a3a3a) + תכלת קר (#0EA5E9, #38BDF8) על רקע לבן/אפור בהיר (#F5F5F5)
- **טיפוגרפיה**: Heebo / Assistant - גופן עברי מודרני, כותרות Bold גדולות, טקסט Regular נקי
- **שפה ויזואלית**: RTL מלא, קווים חדים, פינות מינימליות, אייקונים תעשייתיים (lucide-react), טקסטורות עדינות של בטון/מתכת ברקעים, אנימציות עדינות בגלילה
- **תחושה**: מקצועיות תעשייתית, אמינות, קור (מתאים לעולם הקירור)

## 🌐 צד לקוח (אתר ציבורי)

### עמוד בית
- **חלון ראווה (Hero)** רחב עם תמונת רקע תעשייתית של דלת קירור / חדר קירור, כותרת ראשית גדולה ("פתרונות קירור תעשייתיים מובילים"), תת-כותרת וכפתור CTA לקטלוג
- **רצועת ערכים**: 3-4 אייקונים עם נקודות חוזק (איכות, אמינות, שירות 24/7, ניסיון)
- **הצגת קטגוריות מובילות**: דלתות קירור / חדרי קירור / שירות
- **תצוגה מקדימה של מוצרים נבחרים** מהקטלוג
- **קריאה לפעולה** ליצירת קשר

### עמוד קטלוג דלתות קירור
- **חיפוש חופשי** + **סינון לפי קטגוריה**
- **רשת מוצרים רספונסיבית** (כרטיסים עם תמונה, שם, תיאור קצר)
- **לחיצה על מוצר** → עמוד מוצר מלא עם גלריית תמונות, תיאור מפורט, מפרט טכני, וכפתור "צור קשר לפרטים נוספים"

### עמוד אודות + יצירת קשר (משולב)
- **חלק אודות**: סיפור החברה, ניסיון, ערכים, תחומי פעילות (דלתות קירור + שירות חדרי קירור)
- **טופס יצירת קשר פשוט**: שם, טלפון, אימייל, הודעה - נשמר במסד הנתונים ומופיע בפאנל הניהול
- **פרטי קשר**: טלפון, כתובת, שעות פעילות, מפה (אופציונלי)

### Header & Footer
- **Header**: לוגו, ניווט (בית / קטלוג / אודות ויצירת קשר), כפתור CTA "צור קשר"
- **Footer**: פרטי קשר, ניווט מהיר, זכויות יוצרים

## 🔐 פאנל ניהול (admin בלבד)

### התחברות
- מסך התחברות באימייל וסיסמה
- גישה רק למשתמשים עם הרשאת `admin` (RBAC מאובטח)

### ניהול מוצרים (CRUD)
- רשימת כל המוצרים עם חיפוש וסינון
- הוספת מוצר חדש: שם, קטגוריה, תיאור קצר, תיאור מפורט, מפרט, העלאת תמונות מרובות
- עריכה ומחיקה של מוצרים קיימים

### ניהול קטגוריות
- הוספה / עריכה / מחיקה של קטגוריות
- שם קטגוריה + תיאור + סדר תצוגה

### ניהול פניות
- רשימת כל הפניות שהתקבלו מטופס יצירת קשר
- סימון כ"טופלה" / מחיקה
- מיון לפי תאריך

### ניהול מנהלים
- מנהל קיים יכול להוסיף מנהלים נוספים (לפי אימייל)

## 🛠️ תשתית טכנית (Lovable Cloud / Supabase)

### טבלאות במסד הנתונים
- **categories**: id, name, description, display_order
- **products**: id, name, category_id, short_description, full_description, specifications, images[], created_at
- **contact_messages**: id, name, phone, email, message, is_handled, created_at
- **profiles**: id (FK to auth.users), email, full_name
- **user_roles**: id, user_id, role (enum: admin) - טבלה נפרדת לאבטחה מקסימלית

### אבטחה (RLS)
- **קריאה ציבורית** ל-products ו-categories (כולם רואים את הקטלוג)
- **כתיבה רק למנהלים** (products, categories) דרך פונקציית `has_role()` עם SECURITY DEFINER
- **contact_messages**: כל אחד יכול ליצור (insert), רק מנהלים יכולים לקרוא/לעדכן
- **profiles**: כל משתמש רואה ועורך רק את עצמו

### אחסון תמונות (Supabase Storage)
- Bucket ציבורי `product-images` להעלאת תמונות מוצרים
